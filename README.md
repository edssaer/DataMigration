# 数据库迁移工具

## 项目简介

这是一个可视化的数据库表迁移工具，使用Flask作为后端，Electron作为前端，支持在不同数据库之间进行数据迁移。

## 功能特性

- **数据源管理**：添加、编辑、删除数据源
- **迁移任务管理**：创建、编辑、删除迁移任务
- **数据迁移**：支持不同的表存在策略（插入、清空、重命名）
- **字段映射**：自动映射同名字段，支持手动映射
- **多线程处理**：高效处理大数量级数据迁移
- **响应式设计**：现代化的用户界面，支持不同屏幕尺寸

## 技术栈

- **后端**：Python, Flask
- **前端**：HTML, CSS, JavaScript, Tailwind CSS
- **打包**：PyInstaller, Electron

## 安装说明

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行开发服务器

```bash
python app.py
```

### 3. 构建桌面应用

```bash
# 构建服务器可执行文件
python -m PyInstaller data_migration_server.spec

# 构建Electron应用
cd electron
npm install
npm run build
```

## 使用说明

1. **添加数据源**：在数据源管理页面添加源数据库和目标数据库
2. **创建迁移任务**：在迁移任务管理页面创建新的迁移任务
3. **配置迁移参数**：设置源查询、目标表名、表结构等参数
4. **执行迁移**：点击执行按钮开始数据迁移
5. **查看迁移结果**：查看迁移状态和结果

## 配置说明

- **默认端口**：5000
- **数据库支持**：MySQL
- **配置文件**：`.env` 文件用于存储环境变量

## 注意事项

- 确保源数据库和目标数据库都可以正常访问
- 对于大数据量迁移，建议使用多线程模式
- 迁移前请备份目标数据库中的数据
